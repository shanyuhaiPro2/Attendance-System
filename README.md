# 学生信息管理系统
## 基本功能：
### 添加学生
1. 函数：void addstudent();
2. 功能：添加学生信息（学生信息结构体数组里加）
3. 示例：（终端显示内容）

学号：1

姓名：XX

班级：计科 2507

性别：男/女（同上）
 
### 删除学生或某条考勤记录 
1. 函数：void delect_student();
2. 功能：删除某个学生同时要删掉他的所有考勤记录或者删除某条考勤记录，删之前先显
示出所有学生的信息或者所有考勤记录

### 显示所有学生
1. 函数：void list_student();
2. 功能：展示当前所有学生的基本信息，即学生信息结构体里的内容
3. 示例：（终端显示内容）

学号：1

姓名：XX

班级：计科 2507

性别：男/女

出勤率：100%

......
共计 n 名学生

### 查询学生
1. 函数：void searchstudent();
2. 功能：查询某个学生的信息，可分为学号查询和姓名查询

### 记录考勤
1. 函数：void record_attend();
2. 功能：添加考勤记录
3. 示例：

请输入学号：1

请输入日期：2025-12-12

请输入课程名称：数据结构

请输入考勤状态：
1. 出勤
2. 缺勤
3. 迟到
4. 请假

请输入：

......

### 查看考勤 
1. 函数：void view_attend();
2. 功能：显示所有考勤记录或按学生信息查个人的或者按课程名称查或者按日期查
3. 示例：

1.显示全部考勤情况

2.查某个学生考勤情况

3.查某个课程考勤情况

4.查某天考勤情况

请输入：1

学号 姓名 日期 课程 状态

1 XX 2025-12-12 高数 出勤

2 AA 2025-10-19 c语言 请假

共 2 条考勤记录

请输入：2

学号 姓名 日期 课程 状态

1 XX 2025-12-12 高数 出勤

2 XX 2025-10-19 c 语言 请假

以此类推

### 考勤统计 
1. 函数：void count_attend();
2. 功能：统计所有考勤状态
3. 示例：

总记录数：5

出勤：3

请假：1

缺勤：0

迟到：1

出勤率：100%
